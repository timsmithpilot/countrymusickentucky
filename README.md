# Country Music Kentucky — Website v2.0

Official website for **The Band of Brothers**.

## Version 2 includes

- Transparent Band of Brothers logo
- Mobile-first hero layout
- Existing musician bios, with a subtle Marine Corps Reserve note for Matt
- “Three by blood. One by the road. All country.”
- Correct expandable artist and song list
- Booking email: timsmithpilot@gmail.com
- Individual photo positioning, including Ben’s corrected crop
- Improved motion, accessibility and iPhone spacing

## Main files

- `index.html` — page content
- `style.css` — responsive layout and visual design
- `script.js` — panels and setlist accordion
- `images/` — hero, musician photos and logo

## Preview

Open `index.html` using Live Server in VS Code.

## Publish

1. Save all changes in VS Code.
2. Open GitHub Desktop.
3. Enter a summary such as `Publish website version 2`.
4. Click **Commit to main**.
5. Click **Push origin**.
6. Cloudflare will publish the update automatically.
