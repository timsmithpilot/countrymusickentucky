# Country Music Kentucky

Official website for **The Band of Brothers**.

## Main files

- `index.html` — page content
- `style.css` — layout, colors and responsive design
- `script.js` — interactive panels
- `images/` — website photographs

## Publish an update

1. Save changes in VS Code.
2. Open GitHub Desktop.
3. Enter a short summary.
4. Click **Commit to main**.
5. Click **Push origin**.
6. Cloudflare publishes the update automatically.
